# CDH Maker — Portafolio de servicios

Sitio web del portafolio de servicios de **CDH Maker** (Henry Taborda): desarrollo web y software, fabricación digital (impresión 3D, corte láser, CNC), electrónica e IoT, y diseño/consultoría técnica.

## Estructura

```
public/          # Sitio estático (HTML, CSS, JS)
firebase.json    # Configuración de Firebase Hosting
```

## Desarrollo local

Abre `public/index.html` en el navegador, o sirve la carpeta:

```bash
firebase serve
```

## Despliegue

```bash
firebase deploy --only hosting
```
